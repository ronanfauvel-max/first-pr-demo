# First PR Demo

Petit dépôt créé pour pratiquer le flux de Pull Request avec Claude Code.

## Étapes du flux

1. Cloner le dépôt
2. Créer une branche
3. Faire un changement
4. Ouvrir une Pull Request
5. Faire relire et fusionner la Pull Request
